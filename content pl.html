Programming Languages

  1. Python

    Year of Creation: 1991

    Description: Python is a versatile, high-level programming language known for its simplicity and readability. It supports multiple programming paradigms and has a vast ecosystem of libraries and frameworks.

  2. Java

    3. Year of Creation:1995

    Description:Java is a general-purpose programming language designed to be platform-independent. It is widely used for developing enterprise-level applications and has a strong focus on object-oriented programming.

  3. C

    4. Year of Creation:1972

    Description: C is a low-level programming language known for its efficiency and direct hardware access. It provides a powerful set of features for system programming and is often used in developing operating systems and embedded systems.

  4. JavaScript

    5. Year of Creation:1995

    Description: JavaScript is a high-level scripting language primarily used for web development. It enables interactive and dynamic web content, and it runs on the client-side as well as the server-side with the help of frameworks like Node.js.
